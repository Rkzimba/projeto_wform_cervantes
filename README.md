# Projeto Cervantes - Windows Forms

O Projeto Cervantes é uma aplicação desktop criada para gerenciar dados de forma estruturada através de uma interface gráfica intuitiva.  
O sistema permite operações de cadastro, consulta, atualização e exclusão de registros (CRUD), utilizando banco de dados relacional.

Este projeto foi desenvolvido com foco em:
- Organização de código
- Manipulação de dados
- Integração com banco SQL
- Estruturação de aplicação desktop

## 🛠 Tecnologias Utilizadas

- C#
- .NET
- Windows Forms
- SQL
  
## ⚙️ Funcionalidades

- Cadastro de registros
- Atualização de dados
- Exclusão de registros
- Consulta e listagem
- Integração com banco de dados relacional
